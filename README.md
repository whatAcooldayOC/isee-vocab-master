# ISEE Vocab Master

A mobile-friendly web app for studying ISEE vocabulary.

## Features
- **Daily Words**: Learn new words with definitions and ISEE-specific synonyms.
- **Synonym Match**: Practice identifying synonyms (ISEE Section 1 style).
- **Sentence Completion**: Practice using words in context (ISEE Section 2 style).
- **Text-to-Speech**: Click the speaker icon to hear the pronunciation.

## Usage
Simply open `index.html` in any web browser. 

## Tech Stack
- Single-file HTML/CSS/JS (No dependencies)
- Responsive design for mobile and desktop
